# ❌⭕ XO Challenge JS

## 📘 Overview
XO Challenge JS is a simple web-based **Tic Tac Toe game** developed using **HTML, CSS, and JavaScript**.  
The project demonstrates how basic front-end technologies can be used together to build an interactive browser game.

The game allows two players to take turns placing **X and O** on a 3×3 grid until one player wins or the game ends in a draw.

## 🎯 Project Objectives
- Practice JavaScript logic and DOM manipulation
- Understand event handling in web applications
- Implement game logic using front-end technologies
- Improve problem-solving skills through game development

## 🛠️ Technologies Used
- **HTML** – Structure of the game board
- **CSS** – Styling and layout of the interface
- **JavaScript** – Game logic and user interaction

## 🎮 Game Features
- Interactive 3×3 Tic Tac Toe board
- Two-player gameplay
- Win and draw detection
- Simple and responsive interface

## 🚀 Future Improvements
Possible future enhancements include:
- Player score tracking
- Restart / reset game button
- Single-player mode with AI
- Improved UI and animations
